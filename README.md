# Shadharat (Fragments) 📝
Shadharat is a minimalist, web-based platform designed for authors and thinkers to share their literary creations. The project balances a classical "English Cottage" aesthetic with modern web functionality, providing a distraction-free environment for both long-form essays and short snippets.

# Features ✨
* Dynamic Filtering: Toggle between "All," "Literary Texts," and "Snippets" instantly using Vanilla JavaScript.
* Dual-Mode Content Creator: A custom modal-based editor that adjusts its layout based on the chosen writing format.
* Sophisticated UI: A responsive, high-fidelity design featuring:
 - Classical typography for an editorial feel.
 - Interactive profile dropdown menu.
 - Themed sidebars for trending topics.
* Interactive Feedback: Subtle CSS animations and hover effects for a premium user experience.

# Technical Stack 🛠️ 
* Frontend: HTML5, CSS3 (Flexbox, Grid, Keyframes).
* Scripting: Vanilla JavaScript (DOM Manipulation, State Handling).
* Typography: Optimized Arabic/English fonts (Amiri & Noto Kufi Arabic).
