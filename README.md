## 💡 About

**Shadharat** (Arabic for "golden fragments") is a front-end UI concept for an Arabic literary platform where writers can publish long-form literary essays and short-form snippets in an elegant reading experience built around Arabic typography.

Built as a hands-on learning project to practice UI development fundamentals, with a focus on RTL (right-to-left) layouts, component design, and DOM interaction.

---

## ✨ Features

- **Content Filtering** — Smooth tab-based filtering between literary posts and snippets
- **Write Modal** — Full-screen modal with post-type selection and editor
- **User Profile Dropdown** — Animated dropdown showing user stats and navigation
- **Arabic-first Design** — Full RTL support with Amiri & Noto Kufi Arabic fonts
- **Hover Interactions** — Card lift effects, like button states, nav highlights
- **CSS Animations** — Fade-in and slide-up effects for modals and dropdowns
